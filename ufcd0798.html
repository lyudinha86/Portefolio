UFCD-0798–Utilitários

Formadora: Célia Maria Pinto Duarte

UFCD-0798-Portefólio-Reflexivo de Aprendizagens



Trabalhos realizados

Tarefa 1. "Utilitários"

1. Instalar um antivírus à sua escolha e fazer a análise rápida.
2. Apresentar a captura da imagem dos resultados.
3. Instalar malwarebytes e fazer uma verificação rápida capturando a imagem que deve colocar.
4. Instalar software de compressão de ficheiros (zip like) e mostrar uma imagem do software em funcionamento.

Tarefa 1.


Tarefa 2. "Gestão de discos e compressão de ficheiros."

1. Desinstalar o antivírus instalado na ficha anterior.
2. Desinstalar o malwarebytes.
3. Verificar o estado dos seus discos no Gestor de Discos do Windows
4. Instalar o software AOMEI Partition Assistant - https://www.diskpart.com/thanks-install-pa-standard.html
5. Comparar a informação apresentada por este software com a do Gestor de Discos.
6. Instalar um browser – Chrome, Firefox ou Brave, por exemplo.
7. Localize um software para conversão de ficheiros PDF em formatos Word, Excel ou Powerpoint que permita a conversão de modo gratuito, mesmo que de tamanho limitado. Faça a conversão de um ficheiro em PDF para um dos formatos indicados.


Tarefa 2.


Tarefa 4. "Gestores de dados e outros"

1. Faça o download do SysInternals Suite e apresente uma captura de imagem do TCPVIEW em funcionamento.
2. Descreva brevemente a utilidade do software PuTTy e compare com outros equivalentes, se encontrar.
3. Descreva a utilidade do software WinSCP e compare com outros equivalentes, se encontrar.

Tarefa 4.


Tarefa 5. "Gestores de e-mail, imagens e comunicações."

1. Faça o download do Mozzila Thunderbird Portable e apresente uma imagem da configuração de uma conta de correio.
2. Pesquise por um software de Gestão de Imagens para gerir as imagens no seu computador. Tente encontrar uma aplicação gratuita na Microsoft Store ou uma aplicação Portable. Compare com os chamados Gestores de Bancos de Imagens existentes na Internet.
3. Instale a aplicação Slack a partir da Microsoft Store, e aceite o convite enviado pela formadora para a equipa de Cobol.
4. Apresente exemplo(s), à sua escolha, de utilitários ou Mobile Apps, que considere úteis e não tenham sido abordados na unidade.

Tarefa 5.
